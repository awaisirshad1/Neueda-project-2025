# **App Name**: Mortgage Maestro

## Core Features:

- Mortgage Input: Input fields for loan amount, interest rate, loan term, and down payment.
- Mortgage Calculation: Calculates monthly payment, total interest paid, and total amount paid over the life of the loan.
- Mortgage Visualization: Generates interactive charts displaying the principal vs. interest paid over time.
- Scenario Comparison: Allows users to compare different loan scenarios side-by-side.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) for trust and reliability.
- Background color: Light Blue (#E8EAF6).
- Accent color: Teal (#009688) to highlight key data and interactive elements.
- Body and headline font: 'Inter', sans-serif, for a modern, objective look. Use for headlines and body.
- Use clean, simple line icons to represent different loan aspects and settings.
- A clean, card-based layout for displaying information in a structured and digestible manner.
- Smooth transitions and animations when updating charts and calculations.
